# supabase-java
A Java client for [Supabase](https://supabase.com/).

### PostgREST
Supports basic operations: INSERT, UPSERT, DELETE.

Most of the Horizontal Filtering features are also implemented.

#### Not implemented yet
- Full Text search
- Casting
- Ordering
- Limits and Pagination
- Count
- Singular or Plural
- Custom Queries
- Stored Procedures
- Binary Output
- OpenAPI Support
